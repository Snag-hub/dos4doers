# Getting Started with DOs 4 DOERs

DOs 4 DOERs is now a focused capture-and-remind workflow: save useful content, read in clean mode, and get reminded at the right time.

---

## 1. Create Your Account

1. Open the app and sign up with email, Google, or GitHub.
2. Verify your account if prompted.
3. Sign in and open your inbox.

---

## 2. Understand the Core Areas

- `Inbox`: newly saved links.
- `Favorites`: starred links you want quick access to.
- `Archive`: completed reading references.
- `Trash`: deleted links (recoverable before cleanup).
- `Settings`: reminders, notifications, API token, data export.

---

## 3. Save Your First Item

### Web app
1. Use the save URL action.
2. Paste a link and confirm.
3. Metadata and reader content are extracted automatically.

### Mobile share
1. Open any app/site with a URL.
2. Tap Share.
3. Choose DOs 4 DOERs.

### Browser extension
Use the extension popup to save the active tab in one click.

---

## 4. Install and Connect the Extension

1. In Settings, generate an API token.
2. Install extension build for your browser.
3. Paste token into extension settings and save.
4. Test by saving the current tab.

See `docs/EXTENSION_GUIDE.md` for full install details.

---

## 5. Read and Organize

- Open an item in reader mode when available.
- Use statuses to keep queue clean: inbox -> reading -> archive.
- Star important content to keep it in Favorites.

---

## 6. Configure Reminders

Set reminders in two ways:
- Item-specific reminder: tied to a saved link.
- General reminder: independent reminder from Settings.

Reminder options:
- One-time schedule
- Recurring schedule (`daily`, `weekly`, `monthly`)
- Snooze from notification actions

Configure channels in Settings:
- Email digest and reminder emails
- Push notifications

---

## 7. Recommended First-Day Setup

1. Save 3-5 links using extension/share.
2. Archive one item after reading.
3. Create one item reminder and one general reminder.
4. Enable push notifications.
5. Export your data once to validate backup flow.
