## 1. Create Your Account

1. Visit [DOs 4 DOERs.snagdev.in](https://DOs 4 DOERs.snagdev.in)
2. Click **"Sign Up"** in the top right
3. Choose your sign-up method:
   - Email and password
   - Google
   - GitHub
4. Verify your email address (if using email signup)
5. You're in! 🎉

## 2. Take a Quick Tour

Once logged in, you'll see the main dashboard with several sections:

### 📥 **Inbox**
Your main feed of saved items. Everything you save lands here first.

### ⭐ **Favorites**
Items you've starred for quick access.

### 📚 **Archive**
Items you've read and want to keep for reference.

### 📅 **Timeline**
Visual calendar view of your saved content.

### ✅ **Tasks**
Your to-do list with due dates and priorities.

### 📝 **Notes**
Standalone notes or notes attached to saved items.

### 🗑️ **Trash**
Deleted items (recoverable for 30 days).

## 3. Save Your First Item

There are several ways to save content:

### Method 1: Manual Save (Web App)
1. Click the **"+"** button in the top navigation
2. Paste any URL
3. Click **"Save"**
4. DOs 4 DOERs will automatically extract:
   - Title
   - Description
   - Featured image
   - Full article content (if available)

### Method 2: Share to DOs 4 DOERs (Mobile)
1. Open any app with a share button
2. Tap **Share**
3. Select **"DOs 4 DOERs"** from the share menu
4. The item is saved instantly!

### Method 3: Browser Extension (Recommended)
See [Step 4](#4-install-the-browser-extension) below.

## 4. Install the Browser Extension

The browser extension is the fastest way to save content while browsing.

### Quick Setup

1. **Get Your API Token**
   - Go to **Settings** → **Developer Access**
   - Click **"Generate Token"**
   - Copy the token (you'll need it in step 3)

2. **Download the Extension**
   - Chrome/Edge: Download from Settings → **Browser Extension**
   - Or visit the [Chrome Web Store](#) (coming soon)

3. **Install**
   - **Chrome/Edge**:
     - Extract the downloaded `.zip` file
     - Go to `chrome://extensions/`
     - Enable **"Developer mode"** (top right)
     - Click **"Load unpacked"**
     - Select the extracted folder
   
   - **Firefox**:
     - Extract the downloaded `.xpi` file
     - Go to `about:debugging#/runtime/this-firefox`
     - Click **"Load Temporary Add-on"**
     - Select `manifest.json`

4. **Configure**
   - Click the extension icon in your browser toolbar
   - Paste your API token
   - Click **"Save"**

5. **Start Saving!**
   - Click the extension icon on any page
   - Or right-click → **"Save to DOs 4 DOERs"**
   - Or use keyboard shortcut: `Ctrl+Shift+S` (Windows/Linux) or `Cmd+Shift+S` (Mac)

📖 **Need help?** See the [Complete Extension Guide](../EXTENSION_GUIDE.md)

## 5. Organize Your Content

### Add Tags
1. Open any saved item
2. Click **"Edit"**
3. Type tag names (e.g., "work", "recipes", "tech")
4. Tags help you filter and find content later

### Create Notes
1. Open any saved item
2. Click **"Add Note"**
3. Write your thoughts, highlights, or summaries
4. Notes are searchable and sync across devices

### Mark as Favorite
- Click the **star icon** on any item to add it to Favorites
- Great for items you reference frequently

### Archive When Done
- Click **"Archive"** after reading an item
- Keeps your inbox clean while preserving content

## 6. Set Up Reminders

Never forget to read important articles!

### Set a Reminder
1. Open any saved item
2. Click the **bell icon**
3. Choose when to be reminded:
   - Specific date and time
   - Recurring (daily, weekly, monthly)
4. Click **"Set Reminder"**

### Notification Preferences
Go to **Settings** → **Preferences** to configure:
- **Email Notifications**: Daily summaries and reminders
- **Push Notifications**: Real-time alerts on this device
